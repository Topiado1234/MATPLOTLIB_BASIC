# MATPLOTLIB_BASIC

A beginner-friendly Python project for learning and practicing data visualization using Matplotlib.

This repository covers the fundamentals of plotting charts, customizing visuals, and understanding how data visualization works in Python.

---

## Project Overview

The goal of this project is to help beginners understand:

- Basic plotting with Matplotlib
- Creating line charts, bar charts, pie charts, and scatter plots
- Customizing graph titles, labels, colors, and styles
- Understanding axes, legends, and grids
- Building simple data visualizations for data analysis projects

Matplotlib is one of the most widely used Python libraries for data visualization and is commonly used in data analysis, machine learning, and business intelligence workflows.

## Example Usage

```python
import matplotlib.pyplot as plt

x = [1, 2, 3, 4]
y = [10, 20, 25, 30]

plt.plot(x, y)

plt.title("Simple Line Chart")
plt.xlabel("X Axis")
plt.ylabel("Y Axis")

plt.show()
```
